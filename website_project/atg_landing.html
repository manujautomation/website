Ground Infrastructure

Purpose-Built ATG Tower Network

Our ground stations are not repurposed cell towers. Each site is engineered with skyward-facing directional antennas optimized for aircraft elevation angles, delivering consistent signal at cruise altitude.

  • Sector antennas tuned for 3°–30° elevation angles
  • 2–4 BBUs per site, 3 RRH sectors per BBU
  • Fiber-connected backhaul to regional packet core nodes
  • Remote monitoring via OAM — zero truck-roll maintenance
